# Zihao-YanFullStackBankingApplication

## Getting Started

Install dependencies,
```
node:v12.22.12
```

```bash
cd client
npm install
cd server
npm install
```

Start the dev server,

```
docker pull mongo
```

```bash
cd client 
npm run start
cd server
npm run start
```

build 

```
cd client
npm run build
```
